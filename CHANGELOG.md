# Changelog

All notable changes to the TaaS Node Ecosystem (Truth Node, Challenger) will be documented in this file.

## [1.0.5] - 2026-02-19

### Released
- **Truth Node v1.0.5**:
    - **Thick Node Architecture**: Local execution enabled via embedded engine.
    - **ZK Stub**: Preliminary ZK proof generation support.
    - **Blinded Proxy**: Support for keyless operation via Gateway.
    - **Docker Distribution**: Production-ready `Dockerfile` with source code stripping.

### Added
- **Connectivity Scripts**: Added `test-connection.ts` for verifying Gateway access.
- **Documentation**: Added `CHANGELOG.md` to `truth-node` package.
